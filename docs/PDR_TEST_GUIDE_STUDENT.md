## 🚀 출석 체크하는 방법

### 1단계: 사이트 접속하기

1. **스마트폰에서 인터넷 브라우저 열기** (크롬, 사파리 등)
2. 주소창에 입력: `https://university-attendance-management-sy.vercel.app`
3. 엔터!

### 2단계: 로그인하기

**학생 계정으로 로그인:**
```
아이디: stu001
비밀번호: stu001
```

### 3단계: 권한 허용하기

처음 사용할 때 다음 권한을 허용해주세요:

- **📍 위치 정보 (GPS)** - 출석 위치 확인용 (필수!)
- **📷 카메라** - QR 코드 스캔용 (필수!)

> **걱정 마세요!** 위치 정보는 출석 시간에만 사용되고, 개인정보는 안전하게 보호됩니다.

### 4단계: QR 코드 스캔하기

1. **"QR 스캔" 버튼** 클릭
2. **교수님이 보여주는 QR 코드**를 카메라로 비추기
3. 자동으로 스캔됨!

### 5단계: 위치 확인 기다리기

- "위치 확인 중..." 메시지가 나옵니다
- 3-10초 정도 기다려주세요
- **✅ 출석 체크 완료!** 메시지가 나오면 성공!

## ✨ 시스템이 자동으로 해주는 것들

출석 체크 후, 시스템이 자동으로 다음을 확인합니다:

### 🏫 강의실 범위 안에 있는지 확인
- **30초마다** 자동으로 위치 확인
- 강의실에서 너무 멀리 가면 경고
- **2번 연속** 범위를 벗어나면 **자동 조퇴 처리**

### 🏢 실내/실외 자동 감지
- **실외** (운동장, 주차장): GPS로 정확하게 위치 추적
- **실내** (강의실, 복도): GPS + 센서를 함께 사용해서 더 정확하게 추적
- 자동으로 환경을 감지해서 최적의 방법으로 추적합니다!

## 🎯 이렇게 사용하세요

### ✅ 올바른 사용법

1. **강의실에 들어가기 전**: 실외에서 QR 스캔 (GPS 신호가 좋음)
2. **출석 후**: 스마트폰을 주머니나 가방에 넣어도 OK
3. **수업 중**: 자동으로 위치 추적 (신경 안 써도 됨)
4. **수업 끝**: 교수님이 종료하면 추적 자동 중지

### ⚠️ 주의사항

**조퇴 처리되는 경우:**
- 강의실 범위를 2번 연속 벗어나면 **자동 조퇴**
- 예: 화장실 갔다오는 건 괜찮지만, 오래 나가 있으면 조퇴 처리될 수 있어요

**이렇게 하지 마세요:**
- ❌ 친구에게 스마트폰 빌려주기 → 대리 출석으로 적발
- ❌ 수업 중 너무 멀리 가기 → 자동 조퇴 처리
- ❌ GPS 껐다 켜기 → 위치 추적 실패

## 💡 자주 묻는 질문 (FAQ)

### Q1. 배터리가 빨리 닳나요?
**A**: 아니요! 1시간 수업에 약 3-5% 정도만 소모됩니다.

### Q2. 데이터 많이 쓰나요?
**A**: 아니요! 30초마다 아주 작은 정보만 보내서, 1시간에 약 1MB 미만 사용합니다.

### Q3. 스마트폰을 주머니에 넣어도 되나요?
**A**:
- QR 스캔할 때: 손에 들어야 함 (카메라 사용)
- 출석 후: 주머니/가방에 넣어도 OK!

### Q4. 실내에서 GPS가 잘 안 잡히는데 괜찮나요?
**A**: 걱정 마세요! 우리 시스템은 GPS가 약할 때 **센서를 함께 사용**해서 정확하게 추적합니다.

### Q5. 화장실 가도 되나요?
**A**: 네! 잠깐 다녀오는 건 괜찮아요. 단, **너무 오래 나가 있으면** (2번 연속 범위 이탈) 조퇴 처리될 수 있으니 주의하세요.

### Q6. 위치 추적이 무서워요. 개인정보는 안전한가요?
**A**:
- 위치는 **수업 시간에만** 추적됩니다
- 수업 종료하면 자동으로 추적 중지
- 위치 정보는 출석 확인용으로만 사용
- 교수님과 본인만 볼 수 있습니다

### Q7. 출석이 안 될 때는 어떻게 하나요?
**A**:
1. 인터넷 연결 확인 (Wi-Fi/데이터)
2. GPS 켜져 있는지 확인
3. 카메라 권한 허용했는지 확인
4. 페이지 새로고침 후 다시 시도
5. 그래도 안 되면 교수님께 말씀드리세요!

## 🔧 문제 해결

### 문제 1: QR 코드가 안 스캔돼요
**해결:**
1. 카메라 초점을 QR 코드에 맞추기
2. 너무 가까우면 멀리, 너무 멀면 가까이
3. 밝기가 어두우면 밝은 곳으로 이동
4. 카메라 렌즈 깨끗한지 확인

### 문제 2: "위치를 찾을 수 없습니다"
**해결:**
1. GPS 켜져 있는지 확인 (설정 > 위치)
2. 건물 밖으로 잠깐 나가서 재시도
3. 비행기 모드 껐다 켜기
4. 스마트폰 재시작

### 문제 3: "권한이 거부되었습니다"
**해결:**
1. 설정 > Safari(크롬) > 위치 > 허용
2. 설정 > Safari(크롬) > 카메라 > 허용
3. 브라우저 껐다 다시 켜기
4. 사이트 다시 접속

### 문제 4: 출석했는데 조퇴 처리됐어요
**원인:**
- 강의실 범위를 2번 연속 벗어남
- 예: 화장실 + 매점 = 너무 오래 나가 있음

**해결:**
- 교수님께 사정 설명
- 다음부턴 빨리 돌아오기!

## 🎓 시스템 작동 원리 (궁금한 사람만)

### GPS-PDR 융합 시스템이 뭔가요?

**쉽게 말하면:**
- **GPS**: 인공위성으로 위치 찾기 (실외에서 정확)
- **PDR**: 스마트폰 센서로 걸음 세기 (실내에서 유용)
- **융합**: 둘을 합쳐서 **실내외 어디서나 정확하게** 위치 추적!

**왜 이렇게 했나요?**
- 실내에서는 GPS 신호가 약함 (벽, 천장 때문)
- 센서만 쓰면 시간 지나면 오차 누적
- **둘을 합치면** 정확도 30-50% 향상! 🎯

**예시:**
```
강의실 밖 (실외):
  → GPS 신호 강함 ✅
  → GPS로 정확하게 추적

강의실 안 (실내):
  → GPS 신호 약함 ⚠️
  → GPS + 센서를 함께 사용
  → 걸음 수와 방향으로 위치 보정
  → 여전히 정확! ✅
```
